# ReqTrack Privacy Policy

Last updated: June 2026

## Overview

ReqTrack is a Confluence Forge application for requirement tracking.

## Data Storage

ReqTrack stores requirement metadata using Atlassian Forge hosted storage and Confluence Cloud.

Stored data may include:

- Requirement IDs
- Requirement versions
- Feature names
- Page identifiers
- Internal status information required for application functionality

## Data Processing

ReqTrack processes data only to provide its functionality inside Confluence.

## Third-Party Sharing

ReqTrack does not sell, share, transfer, or disclose customer data to third parties.

## External Services

ReqTrack does not use external servers, external databases, analytics services, advertising services, or tracking services.

## Data Location

Data is stored within Atlassian-hosted services, including Confluence Cloud and Atlassian Forge.
